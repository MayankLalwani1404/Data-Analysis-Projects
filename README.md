📊 Data Analytics Portfolio Projects

This repository contains my data analytics projects focused on building business-driven dashboards using real-world datasets. The goal of these projects is to present data insights in a way that is easy to understand for both technical and non-technical audiences.

🚀 Projects Included
1. 📈 Sales Performance Dashboard (Tableau)
🔍 Overview

An interactive Tableau dashboard built to analyze business performance across revenue, profit, regions, categories, and products.

🎯 Objectives

Track overall business performance using key KPIs

Analyze monthly revenue trends

Identify top-performing regions and categories

Discover the top 10 revenue-generating products

Enable interactive exploration using filters

📊 Dashboard Features

KPIs

Total Revenue

Total Profit

Total Orders

Visualizations

Monthly Revenue Trend (Line Chart)

Revenue by Region (Bar Chart)

Revenue by Category (Bar Chart)

Top 10 Products by Revenue (Bar Chart)

Interactivity

Filters: Region, Category, Order Date

Click-based filtering (dashboard actions)

2. 👥 HR Analytics / Employee Attrition Dashboard (Tableau)
🔍 Overview

An interactive HR analytics dashboard built using Tableau to analyze employee attrition patterns and identify key factors influencing workforce turnover. The dashboard enables both high-level and detailed analysis of employee behavior to support data-driven HR decision-making.

🎯 Objectives

Analyze the overall employee attrition rate

Identify department-wise and job role-based attrition trends

Understand the impact of salary, overtime, and job satisfaction

Evaluate the effect of promotion gaps on employee retention

Enable interactive exploration of workforce data

📊 Dashboard Features

KPIs

Total Employees

Attrition Count

Attrition Rate

Active Employees

Visualizations

Attrition by Department (Bar Chart)

Attrition by Job Role (Bar Chart)

Impact of Overtime on Attrition

Salary Distribution by Attrition (Box Plot)

Attrition by Job Satisfaction

Attrition vs Promotion Gap

Interactivity

Filters: Department, Job Role, Gender, Education Field

Click-based filtering across all charts (Dashboard Actions)
